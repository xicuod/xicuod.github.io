`vi`/`vim` 是 `visual interface`，是 Linux 中最经典的文本编辑器。同图形化界面中的文本编辑器一样，vi 是命令行下对文本文件进行编辑的绝佳选择。vim 是 vi 的加强版本，兼容 vi 的所有指令，不仅能编辑文本，而且还具有 shell 程序编辑的功能，可以不同颜色的字体来辨别语法的正确性，极大方便了程序的设计和编辑性，以后都使用 vim 作为首选编辑器。

## vi/vim 开始使用

启动 vim 的命令是 `vim file.txt`，现在 vim 开始编辑一个名为 `file.txt` 的文件了，如果文件不存在，vim 会新建一个。由于这是一个新建文件，你会得到一个空的窗口。屏幕看起来会像下面这样：("`#`" 代表当前光标的位置)

```
+‑‑‑‑‑‑‑‑‑‑‑‑‑‑‑‑‑‑‑‑‑‑‑‑‑‑‑‑‑‑‑‑‑‑‑‑‑‑‑+
￨#                                     ￨ 
￨~                                     ￨ 
￨~                                     ￨ 
￨~                                     ￨ 
￨~                                     ￨ 
￨"file.txt" [New file]                 ￨ 
+‑‑‑‑‑‑‑‑‑‑‑‑‑‑‑‑‑‑‑‑‑‑‑‑‑‑‑‑‑‑‑‑‑‑‑‑‑‑‑+ 
```

以波纹线 (`~`) 开头的行表示该行不存在。换句话说，如果 vim 打开的文件不能充满这个显示的屏幕，它就会显示以波纹线开头的行。在屏幕的底部，有一个消息行指示文件名为 `file.txt` 并且说明这是一个新建的文件。这行信息是临时的，可以被新的信息覆盖。

## vi/vim 操作模式

|     模式     |   含义   | 主要功能 |
| :----------: | :------: | :------: |
| `--NORMAL--` | 命令模式 |   导航   |
| `--INSERT--` | 插入模式 |   编辑   |
| `--VISUAL--` | 可视模式 |   选择   |
|     `:`      | 命令模式 | 特殊操作 |

- 命令模式 command mode：命令模式下，所敲的按键编辑器都理解为命令，以命令驱动执行不同的功能，不能自由编辑文本。
- 输入模式 insert mode：也就是所谓的编辑模式、插入模式，可以自由编辑文件内容。
- 底行命令模式 lastline mode：以`:` 开始，通常用于文件的保存、退出。

## vi/vim 切换模式

- 从命令模式切换到插入模式：

  | 输入  |     操作     |
  | :---: | :----------: |
  |  `i`  | 在光标前插入 |
  |  `I`  |  在行首插入  |
  |  `a`  | 在光标后插入 |
  |  `A`  |  在行尾插入  |
  |  `o`  | 在下一行插入 |
  |  `O`  | 在上一行插入 |

- `v` 切换普通、可视模式，`Esc` 返回命令模式

## vi/vim 命令模式

命令模式主要用于导航，即移动光标到想要的位置，该模式下的输入映射表如下：

基本导航：

|                            输入                            |                 导航                 |
| :--------------------------------------------------------: | :----------------------------------: |
|                            `h`                             |               左一字符               |
|                            `j`                             |               下一字符               |
|                            `k`                             |               上一字符               |
|                            `l`                             |               右一字符               |
|                            `w`                             |           下一单词开头字符           |
|                            `b`                             |           上一单词开头字符           |
|                            `e`                             |           下一单词结尾字符           |
|                            `ge`                            |           上一单词结尾字符           |
|                       `O`(`Shift+O`)                       |               行首字符               |
|                       `^`(`Shift+6`)                       |          行首第一个非空字符          |
|                       `$`(`Shift+4`)                       |               行尾字符               |
|                            `gg`                            |            首行的开头字符            |
|                            `G`                             |            末行的开头字符            |
| `PageUp`/`PageDown`或<br>`Shift`+`上箭头`/`Shift`+`下箭头` |               上下翻页               |
|                            `/`                             | 进入查找模式，输入关键字按回车来查找 |
|                            `n`                             |        查找下一个 (查找模式)         |
|                       `N`(`Shift+N`)                       |        查找下一个 (查找模式)         |

高级导航：

|   输入    |           导航           |
| :-------: | :----------------------: |
| `f{char}` |    下一 `{char}` 字符    |
| `F{char}` |    上一 `{char}` 字符    |
| `t{char}` | 下一 `{char}` 的前一字符 |
| `T{char}` | 上一 `{char}` 的后一字符 |
|    `;`    |       重复上次查找       |
|    `,`    |       反向上次查找       |

其他快捷键：

- `dd` 删除光标所在行，`ndd` 删除光标往下 n 行
- `yy` 复制光标所在行，`nyy` 复制光标往下 n 行
- `p` 在光标的下一行粘贴复制的内容
- `u` 撤销，`Ctrl+R` 重做
- `dG` 从当前行开始，向下全部删除，`dgg` 从当前行开始，向上全部删除
- `d$` 从当前光标开始，删除到本行的结尾，`d0` 从当前光标开始，删除到本行的开头

## vi/vim 底行命令模式

按 ESC 确保处于命令模式，然后按`:` 进入底行命令模式。

- `:w` 写入 = 保存，`:q` 退出，`:wq` 保存并退出，`:q!` 强制退出，退出但不保存
- `:set nu` 显示行号，`:set paste` 设置粘贴模式，用于在插入模式中保证 `Ctrl+V` 的粘贴格式
