> [Spring](https://spring.io) 使每个人都能更快、更轻松、更安全地进行 Java 编程。Spring 注重速度、简洁性和生产力，这使它成为世界上最流行的 Java 框架。

Spring 框架家族：

- Spring Boot：以独到的视角构建 Spring 应用程序，让你尽快上手并投入运行。
- Spring Framework：为依赖注入、事务管理、Web 应用程序、数据共享、消息传递等提供核心支持。
- Spring Data：为关系型、非关系型、map-reduce 等数据访问提供一致的方法。
- Spring Cloud：为分布式系统中的常见模式提供了一套工具，对构建和部署微服务非常有用。
- Spring Security：通过全面、可扩展的身份验证和授权支持，为您的应用程序提供保护。
- Spring Session：为管理用户会话信息提供了 API 和实现。
- ……

Spring 发展到今天已经形成了一种开发生态圈，Spring 提供了若干个子项目，每个项目用于完成特定的功能。其中 Spring Framework 是较为基础的框架，其他框架都依赖于它。然而，就开发应用程序而言，Spring Framework 配置繁琐，入门难度大。Spring Boot 简化了 Spring Framework，可以帮助我们非常快速的构建应用程序、简化开发、提高效率。Spring Boot 主要做了两件事，起步依赖和自动配置。
