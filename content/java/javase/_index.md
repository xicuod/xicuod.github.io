---
title: Java 标准版 (Java SE)
---
> Java SE（Java Standard Edition，Java 标准版）是Java技术的核心和基础，是 Java ME 和 Java EE 编程的基础。Java SE 是由 Sun Microsystems 公司于 1995 年 5 月推出的 Java 程序设计语言和 Java 平台的总称。用 Java 实现的 Hot Java 浏览器（支持 Java applet）显示了 Java 的魅力：跨平台、动态的 Web、Internet 计算。从此，Java 被广泛接受并推动了 Web 的迅速发展，常用的浏览器均支持 Java applet。——[百度百科](https://baike.baidu.com/item/JAVA%20SE)
