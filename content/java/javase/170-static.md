title: Java static 关键字
---

`static` 的含义是 “类所有”，行为是 “运行时只执行一次”，那就是**类加载**的那一次。

`static` 的正交实体：静态变量、静态方法、静态代码块等。

- 静态变量 = 类变量：类加载时初始化
- 静态方法 = 类方法：静态方法不能直接访问非静态成员
- 静态代码块 = 静态初始化块：类加载时执行，通常用于初始化静态属性
- 静态嵌套类 = 内部类：不持有外部类的引用，因此可以像顶级类一样创建实例，不依赖于外部类的实例

实例构造时，各函数成员的执行顺序：

1. 静态代码块
2. 匿名代码块
3. 构造方法
